This is one of my first full-stack projects: a **User Control Panel (UCP)** for a **GTA: San Andreas Multiplayer (SA:MP)** server, built in **April 2020** using **Laravel 6/7**.

At the time, Laravel 7 had just launched (March 2020), and Laravel 6 (LTS) was still widely used â€” this project likely used one of the two. It served as a functional, self-hosted dashboard where players could:
- View & manage their in-game account (stats, inventory, progress)
- Submit & track support tickets
- Browse server rules, updates, and staff lists
- (Optional integrations) Link Discord, view live player count, etc.

Designed for **roleplay-focused servers**, the UCP prioritized clean UI, responsiveness, and ease of extension â€” no bloated CMS, no unnecessary plugins.
